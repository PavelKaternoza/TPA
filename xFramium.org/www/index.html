xFramium, an open source testing framework for desktop and mobile web, native, and hybrid apps. Write tests using a simple and elegant XML keyword-driven alternative to coding. Use the same test cases on desktop and mobile web browsers. Take advantage of high-performance parallel execution, advanced caching and intelligent device selection algorithms. Easily connect test data from Excel, XML, JSON, CSV, JDBC and more.

Prefer to code in Java? xFramium also supports Java test code writing using all of the features of xFramium such as the extended page object model.

Get xFramium »

Keyword-driven

Not a seasoned developer? No problem! xFramium offers an XML-based extension that allows for tests to be created without writing any code. Most frameworks require automation experts to continuously write code behind the keywords. xFramium’s configuration-based approach is different.

Elegant reporting

xFramium's web-based reports provide developers and testers with all the details they need to quickly localize an issue. Reporting includes historical trends, user interface performance, environmental test coverage, failure categories, and more. xFramium automatically leverages Perfecto’s new reporting solution with no changes required to your tests.

Samples

Define pages and elements using Xpath, HTML properties, optical character recognition, image identification, etc. 
<model siteName="Google">
  <page name="Search">
    <element name="SEARCH" value="//input[@id='lst-ib']" descriptor="XPATH" />
    <element name="SEARCH_BUTTON" value="//button[@id='tsbb']/div|//div[@id='sblsbb']/button" descriptor="XPATH" />
    <element name="RESULTS" value="//*[@id='rso']/div/div" descriptor="XPATH" />
    <element name="RESULT" value="//*[@id='rso']/div/div[{resultIndex}]//a[contains( @href, '{urlToken}')]" descriptor="XPATH" />
  </page>
  <page name="Home">
    <element name="GET" value="Link( 'html tag:=a' ).WebElement( 'html tag:=span', 'innerText:=.*Cart.*')" descriptor="HTML" />
    <element name="TITLE" value="xFramium" descriptor="V_TEXT" />
  </page>
  <page name="Git">
    <element name="TITLE" value="//strong/a[contains( text(), 'xframium-java' )]" descriptor="XPATH" />
  </page>
</model>
Sample of XML keyword-driven test (no coding needed) 
<test name="Search Web">
  <step name="SEARCH" type="EXISTS" page="Search" />
  <step name="SEARCH" type="SET" page="Search">
    <parameter type="STATIC" value="xFramium" />
  </step>
  <step name="SEARCH_BUTTON" type="CLICK" page="Search" />
  <step name="RESULTS" type="WAIT_FOR" page="Search" />
  <step name="RESULTS" type="EXISTS" page="Search" context="resultCount" />
  <!-- Loop through results -->
  <step name="LOOP_OVER_RESULTS" type="LOOP" page="Search">
    <parameter type="CONTEXT" value="resultCount" />
    <parameter type="STATIC" value="checkResult" />
  </step>
  <step name="GET" type="WAIT_FOR" page="Home" />
  <step name="GET" type="CLICK" page="Home" />
</test>
<function name="checkResult">
  <!-- If link is correct, add it to report then click on it -->
  <step name="RESULT" type="EXISTS" page="Search" failureMode="IGNORE" context="textValue">
    <token name="resultIndex" type="CONTEXT" value="_LOOP.INDEX" />
    <token name="urlToken" type="STATIC" value="xframium.org" />
     <step name="ADD_TO_REPORT" type="REPORT" page="Search">
      <parameter type="STATIC" value="URL Link Name" />
      <parameter type="CONTEXT" value="textValue" />
    </step>
    <step name="RESULT" type="CLICK" page="Search">
      <token name="resultIndex" type="CONTEXT" value="_LOOP.INDEX" />
      <token name="urlToken" type="STATIC" value="xframium.org" />
    </step>
    <step name="BREAK" type="BREAK" page="Search" />
  </step>
</function>
The same test case in Java 
@Test(dataProvider="deviceManager")
public void testSearchPage(TestName testName) throws Exception
{
  // Get a proxied instance to your page implementation
  Search searchPage = (Search) PageManager.instance().createPage(Search.class, getWebDriver());

  searchPage.getElement(Search.SEARCH).setValue("xFramium");
  searchPage.getElement(Search.SEARCH_BUTTON).click();        
  searchPage.getElement(Search.RESULTS).waitFor(30, TimeUnit.SECONDS, WAIT_FOR.VISIBLE, "");

  // Loop through results
  for (int i=0; i<searchPage.getElement(Search.RESULTS).getCount(); i++)
  {
    // If link is correct, add to report and click on it
    if (searchPage.getElement(Search.RESULTS)[i].getAttribute("href").contains("xframium.org"))
    {
      searchPage.getElement(Search.RESULT).click();
      continue;
    }    
  }

  searchPage.getElement(Search.GET).waitFor(30, TimeUnit.SECONDS, WAIT_FOR.VISIBLE, "");
  searchPage.getElement(Search.TITLE).isPresent();
  searchPage.getElement(Search.GET).click();
}
