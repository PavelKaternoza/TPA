Features and Benefits

Easy Adoption

Use the xFramium Quick Start configuration tool to quickly find out of xFramium is right for your organization. Start writing a test within 5 minutes!

Application support

Test native applications on Android and IOS based devices, Web based applications on desktop and mobile device and native Windows applications saving time and money while increasing productivity

End user device support

Run seamlessly on mobile devices and desktop browsers in the cloud, connected to your local PC of via a simulator/emulator...or across all combinations simultaneously

High performance

Employing complex caching algorithms and intelligent device selection ensure that your tests are not only accurate but that they are efficient as well. Test case performance is an often overlooked by important metric

Various usage options

xFramium can easily adapt to all organization and user types by allowing configuration for all components to exist independently. Test cases can be created using a single file, multiple files or can exist in a database or in the cloud.

Multiple language support

xFramium supports various scripting methodologies eliminating the need to choose. You group can standardize on one of the methodologies below, use a combination of any of all of them in the same test suite
  • Keyword-driven based tests using XML
  • Keyword-driven based tests using Excel
  • Keyword-driven based tests using SQL Databases
  • Behavior-driven based tests using Gherkin
  • Java based test using TestNG
  • Java based test using JUnit

Script Analysis

Using the xFramium Debugger, users can analyze scripts steps without any development background. Pause, analyze and record steps information for your entire suite

Chained Devices

For test cases that span multiple device such as sending and receiving calls and SMS messages xFramium has you covered. Add devices on the fly to any test case
